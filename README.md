## Spring Boot Book Search & Community Web Application
# - 프로젝트 소개
  이 프로젝트는 Spring Boot를 기반으로 개발한 웹 애플리케이션으로,
  도서 검색 기능과 사용자 인증, 그리고 게시판 기능을 포함한 커뮤니티 서비스를 제공합니다.

# - 주요 기능
  도서 검색 기능: 키워드를 통한 도서 검색, 제목 기반 검색 기능
  사용자 기능: 회원가입(Sign up), 로그인(Login), 로그아웃(Logout)
  게시판 기능: 글 작성, 글 수정, 댓글 작성, 댓글 삭제

# - 기술 스택
  Java, Spring Boot, MySQL, HTML, CSS, Thymeleaf
